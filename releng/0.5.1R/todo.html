openMSX 0.5.1 Release TODO

Introduction

Here is a page for openMSX 0.5.1 release specific TODO list. It is intended for the openMSX release engineering team and openMSX developers. End-users or for more generic information about openMSX and its development, see openMSX web site.

For general feature request TODO, see here.

For general bug fix TODO, see here.

Most of detailed discussion are made on #openMSX at irc.freenode.net.

Issues

This is a list of open issue that need to be resolved for openMSX 0.5.1-Release. If you have any updates for this list, please e-mail to openmsx-devel@lists.sourceforge.net, or change it yourself on this web page, if you have the proper CVS write access.
