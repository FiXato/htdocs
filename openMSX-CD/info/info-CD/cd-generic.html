What is the openMSX-CD

The openMSX CD is a modified KNOPPIX CD which is intended to allow people to try out the openMSX emulator without the need to install any software on their computer.

Why does the openMSX-CD exist

When the openMSX project was started all the main developers used the Linux OS as their main development environment. This ment that most regular users were not able to get aqcuanted with openMSX since there was no binary for other OSes available.

To resolve this situation the idea of a life Linux CD was born.
Due to the growing popularity of the Knoppix and it's faboulus hardware autodetection, it was decided that this should be the base for a the openMSX-CD.

Since we now also officially support Win32, the openMSX-CD also contains a Catapult and openMSX version for win32 platforms. These are custom versions for the openMSX-CD so that you do not need to install openMSX on the machine. As an extra a special loader for the Cd was produced so that a nice autorun feature on Win32 platforms could be made possible

features of this release

  • A collection of freely available software can be found in the openMSX/SOFTWARE directory.
  • win32 features
    • special no-install-needed versions of openMSX and catapult
  • Linux
    • Extra boot options to immediately start a certain MSX machine
    • Auto hardware detection and configuration
    • Several splash screens during boot (depened on the day of the month)
    • Background music as soon as the soundcard is configured
    • An enhanced menu with a custom database for software selection
    • Autorun feature with random software selection in the menu for demo purposes
