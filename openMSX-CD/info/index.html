Welcome to the openMSX CD

release 3.1

This CD contains the openMSX emulator, a collection of softwares and a stripped down version of Knoppix.

More info is available on the following topics:
  • the MSX system
  • the openMSX project
  • the openMSX emulator
  • the openMSX-CD

There is a local copy of the openmsx.sf.net site here.
If you want more info about the original Knoppix, the bootable Linux CD on which this CD is based, you can go to it's official page at http://www.knopper.net/knoppix/
