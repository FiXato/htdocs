Features

Here are some highlights of the features (some only available in the latest CVS version):

  • support for almost all MSX1 and MSX2 systems and also for MSX2+ (and partially MSXturboR) systems, except for the V9958 VDP
  • support for almost all known cartridge mapper types (also with SRAM) via autodetection or the included ROM database
  • configurable with XML files (separation between preferences and machine layout possible)
  • various renderers available (amongst which an openGL based renderer that makes advantage of hardware acceleration of video cards and has some special effects, see screenshots)
  • full interlacing support, real or deinterlaced
  • overscan support
  • MSX-DOS2 support
  • experimental Sunrise IDE with harddisk emulation
  • "Moonblaster Stereo" support: MSX Music on one channel, MSX Audio on the other
  • pixel accurate rendering: horizontal screensplits are rendered almost correctly (Unknown Reality)
  • console with alpha-blended background
  • real floppy disk controller support (WD2793 fully supported, including formatting); also partial support for TC8566AF FDC, National MB8877A and Microsol based FDCs
  • kanji JIS ROM 1 and 2 support
  • mouse and joystick support
  • experimental JoyNet support (connected to TCP/IP of host OS)
  • tape support via CAS and WAV files (real recorded tape image)
  • always real time timing, with sound running in a seperate thread
  • throttle option to run at maximum speed
  • full screen rendering
  • file and command completion in console
  • auto frameskip
  • support for .xsa diskimages
  • speed setting (in percent)
  • support for Konami's Majutsushi and Synthesizer DAC, SCC and SCC+

openMSX cd

On the MSX fair in Bussum of 2002, a self-booting cd was released that can run on many computers, independent of the OS installed on them. This cd was created to demonstrate openMSX to people who do not have Linux installed or who do not or can not download and build openMSX. More information about it will be added here later.
